当你在a分支开发时，不得不切换到b分支，此时应该保证a分支的代码已全部提交，否则无法切换分支

即使可以切换分支，a分支未提交的代码也会带到b分支，污染当前的b分支

于是就有了git stash命令

git stash // 贮存当前分支未提交的代码

而后，b分支处理完了，可以愉快地切换回a分支，并且把贮存的代码提出来

git stash pop // 提取贮存的代码
