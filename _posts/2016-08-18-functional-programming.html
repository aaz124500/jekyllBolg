在函数式编程中，我们希望看到js的纯函数（相同输入，相同输出）

因为函数的行为取决于外部的系统环境，外部的系统环境越复杂，就使依赖于外部系统环境的函数越复杂

柯里化：传递给函数一部分参数来调用它，让它返回一个函数来处理剩下的参数

这是一种“预加载”的做法，得到一个已经记住这些参数的函数，某种程度上“缓存”起来了这些参数

柯里化会产生“包菜式”代码，引起臃肿：a(b(c()))

所以需要一个“函数组合”的函数，把两个函数（或多个函数）结合在一起:

	var compose = function(a, b) {
		return function(x) {
			return a(b(x));
		}
	}
	var a = x => x+1;
	var b = x => x+2;
	var composeFun = compose(a, b)(2);

point free: 不要命名转瞬即逝的中间变量
